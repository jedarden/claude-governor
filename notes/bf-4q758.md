## What the Block Contains

The Some-Some block is a conditional that executes **only when both** `previous_api_snapshot` and `current_api_snapshot` are `Some` (i.e., both are available).

### Data Structures Involved

1. **`state::PrevUsageSnapshot`** (state.rs:134-140)
   - `taken_at: DateTime<Utc>` - When snapshot was taken
   - `five_hour_pct: f64` - 5-hour utilization percentage
   - `seven_day_pct: f64` - 7-day utilization percentage
   - `seven_day_sonnet_pct: f64` - 7-day Sonnet utilization percentage

2. **`db::WindowPctSnapshot`** (db.rs:690-697)
   - `five_hour: f64` - 5-hour window utilization
   - `seven_day: f64` - 7-day all-models window utilization
   - `seven_day_sonnet: f64` - 7-day Sonnet window utilization

### Computations Inside the Block

1. **Convert `PrevUsageSnapshot` to `WindowPctSnapshot`**
   - Creates `prev_pct` from the previous snapshot
   - Creates `curr_pct` from the current snapshot

2. **Calculate window percentage deltas**
   - Calls `calculate_window_pct_delta(&prev_pct, &curr_pct)` (governor.rs:776-784)
   - Returns `(delta_5h, delta_7d, delta_7ds)` - three f64 values

3. **Store deltas in governor state**
   - `state.p5h_delta = Some(delta_5h)`
   - `state.p7d_delta = Some(delta_7d)`
   - `state.p7ds_delta = Some(delta_7ds)`

4. **Log the computed deltas**
   - Logs `[governor] {timestamp} computed window deltas: 5h={+/-X.XXX}% 7d={+/-X.XXX}% 7ds={+/-X.XXX}%`

## Purpose

This block calculates the **change in utilization percentages** between two consecutive API snapshots. The deltas represent how much the fleet-wide utilization has changed during the interval between the previous and current snapshots.
