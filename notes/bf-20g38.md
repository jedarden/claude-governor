## Summary

The current implementation:
1. Uses `if let (Some(prev), Some(curr))` pattern matching to extract both snapshots
2. Only computes deltas when BOTH snapshots are available
3. When `prev_snapshot` is None (first poll), it logs a debug message and does nothing
4. Delta fields remain `None` on first poll, and are only set to `Some(f64)` after successful computation
5. The comment in the else branch incorrectly mentions "default (0.0)" when the actual default is `None`
